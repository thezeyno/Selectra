# ✨ Selectra — Personal Decision Assistant

> Struggling to choose between options? Selectra analyzes your preferences and makes the decision for you.

---

## 🇹🇷 Türkçe

### 📌 Proje Hakkında

Selectra, kullanıcı tercihlerini analiz ederek seçenekler arasında en uygun kararı veren bir **kişiselleştirilmiş karar destek sistemidir**.

Kullanıcı; renk, tarz ve bütçe gibi tercihlerini sisteme girer.  
Ardından birden fazla seçenek ekler ve Selectra bu seçenekleri puanlayarak en mantıklı olanı önerir.

---

### 🚀 Özellikler

- 👤 Kullanıcı kayıt ve giriş sistemi  
- 🎨 Kişisel tercih kaydı (renk, tarz, bütçe)  
- 🧠 Akıllı karar motoru  
- ⚖️ Çoklu seçenek karşılaştırma (3 seçenek destekli)  
- ⭐ Görsel puanlama sistemi (yıldızlarla gösterim)  
- 💡 “Neden bu seçim?” açıklaması  
- 📜 Geçmiş kararları kaydetme ve görüntüleme  

---

### 🧪 Nasıl Çalışır?

1. Kullanıcı tercihlerini belirler  
2. Seçenekleri sisteme girer  
3. Selectra her seçeneği puanlar  
4. En yüksek puanı alan seçenek önerilir  
5. Sistem neden o seçimi yaptığını açıklar  

---

### 🛠️ Kullanılan Teknolojiler

- Python
- Streamlit
- SQLite
- Pandas (opsiyonel kullanım)

## 🇬🇧 English

### 📌 About the Project

Selectra is a personalized decision support system that helps users choose the best option based on their preferences.

Users define their preferences (colors, styles, budget), input multiple options, and Selectra evaluates them to recommend the most suitable one.

---

### 🚀 Features

- 👤 User authentication system
- 🎨 Preference-based decision making
- 🧠 Smart scoring engine
- ⚖️ Multi-option comparison (3 options supported)
- ⭐ Visual scoring system (stars)
- 💡 Explanation system (“Why this choice?”)
- 📜 Decision history tracking

---

### 🛠️ Technologies

- Python
- Streamlit
- SQLite

---

git clone https://github.com/thezeyno/Selectra.git
